# 入居問い合わせ・見学進捗管理ツール

4ペイン構成の画面ひな形です。添付モックアップと要件設計書に基づいています。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 画面構成

| ペイン | 役割 |
|--------|------|
| 第1ペイン（左・狭） | 進捗ステータス一覧（件数バッジ付き） |
| 第2ペイン（中） | 候補者リスト（検索・フィルタ・カード選択） |
| 第3ペイン（広） | 候補者詳細情報（基本・相談支援など） |
| 第4ペイン（右・中） | 運用（ステータス操作・次回対応・保留・面談記録） |

## ディレクトリ

```
src/
  components/     # 各ペインのコンポーネント
  data/           # モックデータ・マスタ参照
  types.ts        # 型定義・ステータス定数
  App.tsx         # 4ペインレイアウト
```

## 現状のスコープ

- UIひな形（静的モックデータ、保存・APIなし）
- ステータス選択 → リスト絞り込み → 詳細・運用ペイン連動
- 検索・フィルタ（保留のみ／今日／超過／未設定）の見た目
